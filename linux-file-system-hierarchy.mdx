## The hierarchy

- \
  - `bin` => user binaries
  - `sbin` => system binaries
  - `etc` => configuration files
  - `dev` => device files
  - `proc` => process information
  - `var` => variable files
  - `tmp` => temporary files
  - `usr` => user programs
  - `home` => home directories (can have multiple users)
  - `boot` => boot loader files
  - `lib` => system libraries
  - `opt` => optional applications
  - `mnt` => mount directory
  - `media` => removable devices
  - `srv` => service data
