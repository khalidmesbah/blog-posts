## Why c

- almost all popular languages are built on top of the c language
- 500,000 lines of code powered the mars curiosity rover
- it's the coding language of choice for kernel development
- it's been around for more than 30 years and isn't going anywhere soon
- writing code in c will ultimately tell you how a computer really works
- it's on high demand
- it's portable
