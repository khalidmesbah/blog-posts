## what is pnmp

it is a node package manager like npm and yarn

(link)[https://pnpm.io/]

## what is a node package manager

it is a program to manage node dependencies

## why to use pnpm

because it is 
  - efficient: files inside node_modules are cloned or hard linked from a single content-addressable storage resulting in less storage usage
  - fast: faster than npm and yarn

resulting in a faster, cleaner, and more organized development environment
